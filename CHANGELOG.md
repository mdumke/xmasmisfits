# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.5.0] - 2025-11-27

### Fixed

- Avoid audio initialization warning
- Do not auto-resume manually paused audio

## [0.4.0] - 2025-11-26

### Added

- Audio-visual feedback for locked doors
- Bells for opening doors
- Remember last scroll position
- Make initial position adjustable
- Speaker test

### Changed

- Switch back to 2560x1440 format
- Remove title-sliding

### Fixed

- Space-bar no longer starts scroll

## [0.3.0] - 2025-11-22

### Added

- stop-motion animation
- fullscreen-toggle

### Fixed

- Show content thumbnails while curtain is lifted

## [0.2.0] - 2025-11-21

### Added

- edge-blur overlay
- snow overlay
- play door sound
- play wind ambience sound
- global audio-toggle
- pause audio while browser tab is hidden

### Changed

- switch to larger bg image (3200x1800)
- slide-transition to calendar
